# Manus Sidebar Reference (from user screenshots)

## Desktop Sidebar Structure (Left Panel)
1. **Logo** "manus" with sparkle icon + close (X) button
2. **Credits badge** "6,600 credits" + **Version badge** "v2.0" (sparkle icon)
3. **Search bar** "Search tasks & messages..." with filter icon
4. **TASKS** section header with filter icon
   - "+ New task" button
   - Task list (scrollable, each task has icon + title + ... menu)
   - Tasks show different icons based on type (circle, AI icon, document icon, etc.)
5. **MANUS** section header
   - Analytics (bar chart icon)
   - Memory (brain/globe icon)
   - Projects (folder icon)
   - Library (book/grid icon)
   - Schedules (clock icon)
6. **"Share with a friend"** card - "Get 500 credits each"
7. **User profile** "Michael" with avatar at very bottom

## Mobile Sidebar (Hamburger Menu)
- Same structure as desktop but in overlay panel
- Same sections: TASKS, MANUS, Share, User

## Mobile Bottom Navigation Bar
- **Home** (house icon) - active state filled
- **Tasks** (list icon)
- **Billing** (card/rectangle icon)
- **More** (...) dots icon

## Key Differences from Our Current App
- Our sidebar is missing the TASKS section entirely (search + task list)
- Our sidebar has tasks in the main content area, not the sidebar
- Our mobile bottom nav has different items
- Manus uses section headers "TASKS" and "MANUS" in uppercase
- Manus sidebar is dark theme with subtle borders
