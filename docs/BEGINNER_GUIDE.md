# Manus Next — Beginner's Guide

Welcome to **Manus Next**, your autonomous AI agent platform. This guide walks you through everything you need to get started in under 5 minutes.

---

## 1. First Launch

When you open Manus Next, you will see the **Home screen** — a conversational interface with a text input at the center. This is where all interactions begin.

**To sign in:** Click your avatar or the "Sign In" link in the sidebar. You will be redirected to Manus OAuth. After signing in, your tasks and preferences are saved across sessions.

**To use without signing in:** You can start immediately — tasks will be stored locally in your browser. Sign in later to sync them to the cloud.

---

## 2. Creating Your First Task

Type any request into the input box on the Home screen and press **Enter** (or click the arrow button). Examples:

| What to type | What happens |
|---|---|
| "Research the latest AI agent frameworks" | Agent searches the web, reads pages, and writes a synthesis |
| "Generate an image of a mountain sunset" | Agent creates an AI-generated image |
| "Analyze this data and create a chart" | Agent processes data and produces visualizations |
| "Write a project proposal for a mobile app" | Agent generates a structured document |

You will be taken to the **Task View**, where you can watch the agent work in real time.

---

## 3. Understanding the Task View

The Task View has two main areas:

**Left panel — Chat:** Your conversation with the agent. You will see:
- Your messages (right-aligned)
- Agent responses with streaming text (left-aligned)
- Tool execution steps (collapsible cards showing what the agent is doing)

**Right panel — Workspace** (desktop only): Shows artifacts the agent creates:
- **Browser tab:** Screenshots and web pages the agent visits
- **Code tab:** Code the agent writes or executes
- **Terminal tab:** Command output from code execution
- **Preview tab:** Generated documents, images, and slides

On mobile, the workspace appears below the chat. Tap the workspace tabs to switch between views.

---

## 4. Key Features

### Voice Input
Click the **microphone button** next to the text input. Speak your request, and it will be transcribed and sent automatically. Works on both Home and Task View.

### File Attachments
Click the **paperclip button** to attach files, or drag and drop files directly into the chat. The agent can read documents, analyze images, and process data files.

### Task Management
Your tasks appear in the **sidebar** (left side on desktop, swipe from left on mobile):
- **Search:** Type in the search box to find tasks by title or content
- **Filter:** Use the status tabs (All / Running / Done / Error) to filter
- **Delete:** Hover over a task and click the trash icon
- **Favorite:** Click the star icon to bookmark important tasks

### Keyboard Shortcuts
Press **Cmd+/** (Mac) or **Ctrl+/** (Windows) to see all shortcuts:

| Shortcut | Action |
|---|---|
| Cmd+K | Focus the search/input box |
| Cmd+N | Create a new task |
| Cmd+Shift+S | Toggle the sidebar |
| Escape | Close any open dialog |

---

## 5. Agent Tools

The agent has 14 built-in tools it can use autonomously:

**Research:** Web search, wide research (parallel queries), webpage reading, enhanced browsing
**Creation:** Image generation, document generation, slide creation, design composition
**Analysis:** Data analysis, code execution (sandboxed JavaScript)
**Communication:** Notifications, email
**Media:** Audio transcription, file management

You do not need to tell the agent which tool to use — it decides automatically based on your request. Some tools (like sending email or executing code) require your approval before running.

---

## 6. Settings

Access Settings from the sidebar (gear icon). Key sections:

**General:** Set your default system prompt (instructions the agent always follows), display name, and response preferences (speed vs. quality mode).

**Capabilities:** View the status of all platform features. Toggle capabilities on/off.

**Connectors:** Connect external services (Slack, GitHub, Google, Notion, etc.) to extend what the agent can do.

**Privacy:** Manage saved cookies and data.

---

## 7. Billing

Access the Billing page from the sidebar. Here you can:
- View your current subscription plan
- See task usage statistics
- Subscribe to Manus Next Pro for higher limits
- View payment history

Test payments use the card number `4242 4242 4242 4242` with any future expiry date and any CVC.

---

## 8. Mobile Experience

Manus Next is fully responsive. On mobile:
- **Sidebar:** Swipe from the left edge or tap the menu icon
- **Bottom navigation:** Quick access to Home, Tasks, and Settings
- **Workspace:** Appears below the chat in a tabbed layout
- **Voice:** Tap the mic button for hands-free input

---

## 9. Tips for Best Results

1. **Be specific:** "Research the top 5 AI agent frameworks released in 2025 and compare their architectures" works better than "tell me about AI"
2. **Use follow-ups:** After the agent responds, ask follow-up questions in the same task to refine results
3. **Attach context:** Upload relevant documents or paste URLs to give the agent more context
4. **Use system prompts:** Set a global system prompt in Settings to customize the agent's behavior for your workflow
5. **Try voice:** Voice input is often faster for complex requests

---

## 10. Troubleshooting

| Issue | Solution |
|---|---|
| Agent stops responding | Click "Regenerate" on the last message, or send a follow-up message |
| Voice input not working | Check browser microphone permissions (Settings > Privacy > Microphone) |
| Task not saving | Sign in to enable cloud persistence |
| Slow responses | Switch to Speed mode in Settings > General |
| Page not loading | Refresh the browser (Cmd+R / Ctrl+R) |

For additional help, visit the Settings page or contact support.
