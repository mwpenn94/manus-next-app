# Manus Recording 1 Analysis (730s, 767MB)

## 1. UI Surfaces Visible

- **Left Sidebar:** Persistent navigation hub with Home, Projects, Library, Agent configuration
- **Chat Interface:** Full-width chat for standalone tasks, with message history and file attachments
- **Task Workspace:** Focused workspace when task is clicked, shifts from dashboard to chat view
- **Right Context Panel:** Appears for tasks within a Project to manage project-specific variables
- **Integrated Developer Console:** Full browser DevTools (Edge) running alongside chat showing headless browser execution logs, network traffic, DOM elements
- **Settings & Management Modals:**
  - Account / Usage & Billing: Detailed credit consumption by task, Cloud, AI, Integrations
  - Data Controls: Shared tasks, files, Websites, Apps, Purchased domains, "My Computer"
  - Skills & Connectors: Marketplaces to enable AI capabilities and link third-party APIs
  - Mail Manus: Dedicated email addresses to trigger tasks

## 2. Capabilities Demonstrated

- **Multimodal Input:** File attachments, Skills dropdown, audio recording, screen recording
- **Autonomous Task Execution:** Long-running autonomous operation with progress reporting, rate limit handling, decision-making
- **Artifact Generation & File Management:** ZIP, CSV, Markdown reports; Files modal with type categorization; Batch download
- **Project Organization:** Hierarchical structure (tasks → Library → Projects) with isolated instructions, files, skills
- **Scheduled & Background Tasks:** Automatic scheduling (e.g., "Every 2 hours"), dedicated dashboard

## 3. Pricing and Billing

- **Credit System:** ~900,000 credits balance, granular metering per task (Cloud/AI/Integrations), 300 daily refresh credits
- **Team Plan:** $20/seat/month ($200/year), 4,000 credits/seat/month, SSO, Team analytics, Internal access control
- **Manus Pro:** $200/month, 40,000 credits/month, in-depth research, batch production
- **High-Tier:** $10,000/month = 2,400,000 credits/month; up to $20,000/month = 5,000,000 credits, 30 concurrent + 30 scheduled tasks

## 4. Navigation Patterns

- **Hub-and-Spoke Model:** Left sidebar as persistent hub → Projects, Library, Agent pages
- **Modal-Heavy Configuration:** Settings, billing, personalization, file management all in centered overlay modals
- **Contextual Task Workspaces:** Dashboard → focused workspace on task click; Project tasks get right-hand context panel

## 5. Notable/New Features

- **Integrated Developer Console:** Browser DevTools exposed alongside chat for debugging transparency
- **Knowledge Personalization:** Toggleable complex behavioral rules under Profile → Personalization → Knowledge
- **Mail Manus:** Inbound email (e.g., `mwpeen949@manus.bot`) to trigger workflows with approved senders whitelist
- **My Browser Connector:** Integrations menu shows connector requiring "Folder Access" for local browser/filesystem interaction
